DESCUBRE Site Plan Document

Site Name: DESCUBRE

Reason for Selection:
The name "DESCUBRE" translates to "discover" in English, which aligns with the brand's mission of helping customers discover unique and stylish urban clothing inspired by Mexican culture.

Site Purpose:

The purpose of the DESCUBRE website is to showcase and sell urban clothing that reflects Mexican heritage and contemporary fashion trends. The site will provide information about the brand, feature a product catalog, and offer an easy-to-use shopping experience. Additionally, the site will include a blog section to discuss fashion trends and cultural inspirations behind the designs.

Scenarios:

  • Where can I purchase DESCUBRE's latest collection?
  • How do I find information about the materials and origins of the clothing?

Color Schema:

Primary Color: #FF5733 (Bright Orange) - This color will be used for headings, buttons, and accents.
Secondary Color: #2C3E50 (Dark Blue) - This color will be used for the background, footers, and secondary text.

Typography:

Headings Font: Montserrat, sans-serif - Used for all headings to give a modern and bold look.
Body Font: Roboto, sans-serif - Used for the main body text to ensure readability and a clean appearance.

Wireframe:

Home Page Layout:

Mobile View:

    Header: Logo on the left, menu icon on the right.
    Hero Section: Full-width image with tagline and shop now button.
    Featured Products: Scrollable product carousel.
    About Section: Brief introduction to the brand.
    Blog Preview: Links to latest blog posts.
    Footer: Navigation links, social media icons, contact information.

Desktop View:

    Header: Logo on the left, navigation links in the center, search bar and cart icon on the right.
    Hero Section: Full-width image with tagline and shop now button.
    Featured Products: Grid layout of top products.
    About Section: Brief introduction to the brand with an image.
    Blog Preview: Horizontal scrollable list of latest blog posts.
    Footer: Full-width with navigation links, social media icons, and contact information.

CSS:

Supporting CSS files will include:

  • base.css: For general styles and resets.
  • layout.css: For layout-specific styles including grid and flexbox arrangements.
  • typography.css: For font styles and text-specific styles.
  • colors.css: For color-specific styles based on the selected color schema.

Testing:

The site will be tested using the following tools:

  • HTML Validator: To ensure the HTML is well-formed and standards-compliant.
  • Accessibility Checker: To ensure the site is accessible to all users.
  • Color Contrast Checker: To ensure sufficient contrast between text and background colors.
  • SEO Analysis Tool: To optimize the site for search engines.
  • Performance Analyzer: To check and improve page load times and overall performance.
